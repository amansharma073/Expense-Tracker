# Expense Tracker

A responsive Expense Tracker web application built using HTML, CSS, and JavaScript. It helps users manage income and expenses, track transactions, and monitor their balance in real time.

## Features

* Add income and expense transactions
* Automatic balance calculation
* Real-time income and expense updates
* Transaction history tracking
* Data persistence using LocalStorage
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* LocalStorage API

## How It Works

1. Enter a transaction description.
2. Enter the transaction amount.
3. Select Income or Expense.
4. Click **Add Transaction**.
5. The application updates:

   * Total Balance
   * Total Income
   * Total Expenses
   * Transaction History

All data is stored in LocalStorage, so transactions remain available even after refreshing the page.
